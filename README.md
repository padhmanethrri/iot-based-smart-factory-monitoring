# Smart Factory IoT Monitoring System

## Overview
This project is a dual-node Industrial IoT monitoring and automation system developed using Arduino Uno, PicsimLab simulation, MQTT communication, and ThingsBoard Cloud.

The system monitors real-time factory conditions across two areas:

- Production Line Node
- Warehouse Node

## Features
- Real-time sensor monitoring
- MQTT cloud telemetry
- Dashboard visualization
- Relay automation
- Alarm conditions
- Remote control using RPC
- Multi-node architecture

## Technologies Used
- Arduino Uno
- C++
- MQTT
- ThingsBoard Cloud
- PicsimLab
- Embedded Systems

## Project Structure
- `Production_Line_Node/` → Node 1 source files and connections
- `Warehouse_Node/` → Node 2 source files and connections
